### Task 8: Nested Routes (25 min)
Install `drf-nested-routers` and implement:
```
GET  /api/books/<book_pk>/reviews/         → list all reviews for a book
POST /api/books/<book_pk>/reviews/         → add a review to a book
GET  /api/books/<book_pk>/reviews/<id>/    → get a specific review
```

Model for Review:
```python
class Review(models.Model):
    book = models.ForeignKey(Book, on_delete=models.CASCADE, related_name='reviews')
    reviewer = models.ForeignKey(User, on_delete=models.CASCADE)
    rating = models.IntegerField()  # 1-5
    comment = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)
```
